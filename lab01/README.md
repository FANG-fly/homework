# 实验1(lab01)：Arduino IDE 安装与 ESP32 环境配置

## 实验内容
- 安装 Arduino IDE
- 配置 ESP32 开发板支持
- 安装 CH340 USB 驱动
- 运行基础测试程序

## 实验结果
- 开发板型号：ESP32 Dev Module
- 端口：COM3（根据实际情况填写）
- 上传波特率：921600
- 程序功能：板载 LED 每秒闪烁一次，串口输出 "Hello ESP32!"
